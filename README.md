# Website Rental Motor

Website Rental Motor adalah sistem berbasis web yang digunakan untuk mempermudah proses penyewaan motor secara online serta membantu admin dan petugas dalam mengelola data penyewaan.

---

## Tampilan Website

### Halaman Login

<p align="center">
<img src="images/login.png" width="700">
</p>

Halaman login digunakan oleh pengguna untuk masuk ke dalam sistem dengan memasukkan email dan password yang telah terdaftar.

---

## Admin

### Dashboard Admin

<p align="center">
<img src="images/dashboard-admin.png" width="700">
</p>

Dashboard admin menampilkan informasi utama sistem seperti:

* Total data motor
* Total penyewaan
* Data pelanggan
* Laporan transaksi

Admin juga memiliki akses untuk mengelola:

* Data motor
* Data pelanggan
* Data petugas
* Data transaksi rental

---

## Petugas

### Dashboard Petugas

<p align="center">
<img src="images/dashboard-petugas.png" width="700">
</p>

Dashboard petugas digunakan untuk membantu operasional penyewaan motor seperti:

* Mengelola data penyewaan
* Menginput transaksi rental
* Melihat data pelanggan
* Mengelola pengembalian motor
